Exercice 2 : Jeu du nombre à deviner .

## Description
L'ordinateur génère un nombre aléatoire entre 1 et 100.
Le joueur a 10 essais pour le deviner.
À chaque essai, le jeu indique si le nombre est trop grand ou trop petit.

## Règles
- Nombre aléatoire entre 1 et 100
- Maximum 10 essais
- Indice donné à chaque essai (trop grand / trop petit)
- Victoire si trouvé avant 10 essais, défaite sinon

## Exécution
Ouvrir le projet dans IntelliJ IDEA et lancer `Main.java`
